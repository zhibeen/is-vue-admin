# 生产环境依赖补充

## 阿里云日志服务 SDK

如果需要启用阿里云 SLS 日志上传功能，请安装以下依赖：

```bash
pip install aliyun-log-python-sdk>=0.8.0
```

或添加到 `requirements.txt`:

```
# 阿里云日志服务（生产环境可选）
aliyun-log-python-sdk>=0.8.0
```

## 安装说明

### 开发环境
开发环境默认不需要此依赖，日志会输出到本地文件。

### 生产环境
如果启用 `ENABLE_ALIYUN_SLS=true`，需要安装此依赖：

```bash
# 在后端容器中安装
docker compose exec backend pip install aliyun-log-python-sdk

# 或重新构建镜像
docker compose build backend
```

## 验证安装

```bash
docker compose exec backend python -c "import aliyun.log; print('SLS SDK installed successfully')"
```
